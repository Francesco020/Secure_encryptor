## Project Overview
This project is designed as a **learning and portfolio tool** for cybersecurity and Python programming.
It demonstrates:
- Password-based key derivation (PBKDF2)
- AES symmetric encryption using Fernet
- File handling (binary read/write)
- Command*-line interface (CLI)

> **Note:** This is a mini-project for educational purposes.
